2 · Initiate flow — cross-module via outbox (drift-free)

Moving candidates to Collateral splits the old single transaction. A sync service call would risk "one leg fails" drift, so we use the outbox: Collateral commits candidate state + event atomically; Request creates the Requests via an InboxGuard consumer.
