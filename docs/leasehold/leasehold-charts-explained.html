1. Cumulative PV % คืออะไร (กราฟขวา)

"Cumulative PV %" (เส้นสีม่วงในกราฟ Leasehold Value Composition) คือ ผลรวมสะสมของมูลค่าปัจจุบัน (Present Value) ของรายได้ค่าเช่า โดยคิดเป็นเปอร์เซ็นต์ของมูลค่าปัจจุบันรวมของค่าเช่าทั้งหมดตลอดอายุสัญญาเช่า

แท่งกราฟ 2 ชุด

  • Rental Income (สีส้ม) — ค่าเช่าดิบที่เก็บได้ในแต่ละปี เป็นเงินสดตามจริง ยังไม่ได้คิดลด (discount)
  • Net PV Rental (สีเขียว) — ค่าเช่าปีเดียวกันนั้นที่ถูกคิดลดกลับมาเป็นมูลค่าวันนี้ เงินในอนาคตมีค่าน้อยกว่าเงินวันนี้ แท่งเขียวจึงเตี้ยกว่าแท่งส้ม และช่องว่างถ่างขึ้นเรื่อยๆ เมื่อปีไกลออกไป

เส้นสีม่วง — Cumulative PV %

ตอบคำถามว่า: "มาถึงจุดนี้ของสัญญาเช่า เราสะสมมูลค่าค่าเช่า (ในเงินวันนี้) มาแล้วกี่ % ของทั้งหมด?"

Cumulative PV %(ปีที่ n) = Σ Net PV Rental (ปี 1 ถึง n) ───────────────────────────── Σ Net PV Rental (ทั้งสัญญา)
  • เริ่มใกล้ 0% ในปีที่ 1 แล้วไต่ขึ้นไปถึง 100% ตอนสิ้นสุดสัญญาเช่า — ตามนิยาม เมื่อรวมค่าเช่าคิดลดของทุกปีครบ ก็ได้ครบ 100%
  • เส้นจะชันมากช่วงต้นและแบนราบช่วงท้าย เพราะปีแรกๆ ให้มูลค่าคิดลดก้อนใหญ่ ส่วนปีท้ายๆ (ถูกคิดลดหนัก) เพิ่มแค่นิดเดียว — ความโค้งนี้คือลักษณะเด่นของการคิดลด
Reversion (constant): tooltip ยังแสดง "+ Reversion PV (constant)" ด้วย เป็นคนละก้อน — มูลค่าทรัพย์ที่ได้คืนตอนหมดสัญญา คิดลดมาเป็นเงินวันนี้ ที่เขียนว่า "constant" เพราะเป็นเงินก้อนเดียวตอนจบ ไม่สะสมทีละปีเหมือนค่าเช่า นี่คือเหตุผลที่หัวกราฟเขียน Reversion 95–97% vs Rent 3–5% — มูลค่าเกือบทั้งหมดอยู่ที่การได้ทรัพย์คืน ไม่ใช่ค่าเช่าระหว่างทาง

2. กราฟซ้าย — Collateral Value Over Lease Term

หัวกราฟเขียน "All values in today's PV baht" = ทุกค่าในกราฟนี้เป็นมูลค่าปัจจุบัน (PV วันนี้) ทั้งหมด ไม่ใช่ค่าในอนาคต

เส้นคืออะไร
Total L&B (PV today)
(พื้นที่เขียว ลาดลง)		 มูลค่าปัจจุบันของที่ดิน+อาคารทั้งหมด ถ้าได้คืน ณ ปีนั้นๆ — ในตารางเป็นค่า nominal เท่ากันทุกปี แต่ในกราฟลาดลงเพราะถูกคิดลดกลับมาเป็นเงินวันนี้ (ยิ่งได้คืนช้า มูลค่าวันนี้ยิ่งต่ำ) จุดปลายสุดตรงกับ "Value at Lease Expiry" ในตาราง
Building (PV today)
(ล่างสุด ใกล้ 0)		 มูลค่าปัจจุบันของเฉพาะตัวอาคาร (ซึ่งเสื่อมราคาด้วย) เคสตัวอย่างเป็นที่ดินล้วน ไม่มีอาคาร เส้นนี้จึงแทบติดศูนย์
Encumbered Collateral (PV today)
(เส้นประ)		 มูลค่าหลักประกัน leasehold ที่ผู้ให้กู้เอาคืนได้จริง ณ แต่ละปี = PV ค่าเช่าที่เหลือ + PV reversion (ดูหัวข้อถัดไป)
ถ้าถือกรรมสิทธิ์เต็ม (freehold) มูลค่าจะอยู่ที่จุดเริ่มของพื้นที่เขียว แต่เพราะนี่เป็น leasehold (สิทธิการเช่า) มูลค่าจริงในวันนี้ลดลงเหลือแค่เส้นประ — ส่วนต่างที่หายไปคือ "ราคา" ของการเป็นแค่ผู้เช่า ไม่ใช่เจ้าของตลอดไป

3. Encumbered Collateral (PV today) คืออะไร

"Encumbered Collateral" = หลักประกันที่ติดภาระผูกพัน (ติดสัญญาเช่า) — คือมูลค่าหลักประกันที่ผู้ให้กู้เอาคืนได้จริง โดยคำนึงว่าทรัพย์นี้ติดภาระ leasehold ไม่ใช่กรรมสิทธิ์เต็ม

สูตรจริงในโค้ด (LeaseholdChart.tsx:146):

encumberedCollateralValue = remainingRentPv[i] + reversionPvToday

ก้อนที่ 1 — remainingRentPv[i] : PV ค่าเช่าที่เหลือ

ผลรวมมูลค่าปัจจุบันของค่าเช่าที่ยังเหลืออยู่ ตั้งแต่ปีที่ i ไปจนสิ้นสุดสัญญา (คำนวณโดยไล่จากปีสุดท้ายย้อนกลับมา สะสม netCurrentRentalIncome) — ปียิ่งผ่านไป ค่าเช่าที่เหลือยิ่งน้อย ก้อนนี้จึงหดลงเรื่อยๆ

ก้อนที่ 2 — reversionPvToday : PV ทรัพย์ที่ได้คืน

มูลค่าที่ดิน+อาคารตอนสิ้นสุดสัญญา คิดลดกลับมาเป็นเงินวันนี้ — เป็นค่าคงที่ (constant) ทุกปี

ความหมายรวม: ณ จุดเวลาใดๆ ผู้ให้กู้จะได้คืน = ค่าเช่าที่ยังเก็บได้จากนี้ไป + มูลค่าทรัพย์ที่ได้คืนตอนหมดสัญญา (ทั้งคู่เป็นเงินวันนี้) → เริ่มสูงสุดตอนต้นสัญญา แล้วค่อยๆ ลดลงจนเหลือแค่มูลค่า reversion ตอนหมดสัญญา

4. ตัวอย่างการคำนวณ — 6,955,392.23 มาได้ยังไง

ที่ Year 13.8 (ปีสุดท้าย) ตามสูตร PV(remaining rent) + PV(reversion):

ก้อนที่ 1 — PV(reversion) = 6,946,442.52

reversionPvToday = totalLandAndBuilding(nominal) × pvFactor ปีสุดท้าย = 15,523,279.57 × 0.44748… = 6,946,442.52 (pvFactor ปีสุดท้าย = netRental / rental = 8,949.71 / 20,000 ≈ 0.4475)

ก้อนที่ 2 — PV(remaining rent) = 8,949.71

เพราะเป็นปีสุดท้าย จึงเหลือค่าเช่าแค่งวดเดียว = Net Current Rental Income ของ Year 13.8 = 8,949.71

รวมกัน

6,946,442.52 (PV reversion) + 8,949.71 (PV ค่าเช่าที่เหลือ = งวดสุดท้าย) ───────────── = 6,955,392.23 ✓ ตรงเป๊ะ

5. หมายเหตุ — ส่วนต่าง ~1.47 บาท

มี "มูลค่าทรัพย์คืน" 2 ตัวที่ใกล้กันแต่ไม่เท่ากัน:

ค่าตัวเลขที่มา
Total L&B (PV today) ใน tooltip6,946,442.52nominal × pvFactor งวดสุดท้าย (0.4475) → ใช้ในสูตร Encumbered
Value at Lease Expiry ในตาราง6,946,443.99คิดลดด้วย pvFactor เต็มความละเอียด
ต่างกัน ~1.47 บาท เพราะ pvFactor คนละตัว — reversionPvToday (LeaseholdChart.tsx:104-107) คำนวณ pvFactor จาก netCurrentRentalIncome / rentalIncome (ค่าปัดเศษแล้ว) แทนที่จะใช้ pvFactor ตรงๆ จากตาราง ในเชิงอุดมคติสองค่านี้ควรเท่ากัน หากต้องการความเป๊ะ ควรให้ทั้งสองจุดใช้ pvFactor ตัวเดียวกัน

6. ค่านี้บอกอะไรกับเรา

ตัวเลข ~6.95M (Encumbered Collateral) บอกว่า "หลักประกันนี้มีค่าจริงเท่าไหร่ในวันนี้"

  • ทรัพย์ตามราคาตลาดเต็ม (freehold) = ~15.5M แต่เป็น leasehold มูลค่าจริงวันนี้เหลือแค่ ~6.95M
  • ส่วนต่าง ~8.5M ที่หายไป = "ราคา" ของการเป็นแค่ผู้เช่า ไม่ได้เป็นเจ้าของตลอดไป
  • ประกอบด้วย: ค่าเช่าตลอดสัญญา ~3% + มูลค่าทรัพย์ที่ได้คืน (reversion) ~97% → มูลค่าเกือบทั้งหมดอยู่ที่การได้ทรัพย์คืน ไม่ใช่ค่าเช่า
ข้อควรระวังในการตีความ:
  • เป็นมูลค่าปัจจุบัน (PV today) แล้ว — อย่าเอาไปบวกกับตัวเลข nominal
  • ขึ้นกับ discount rate อย่างมาก — เปลี่ยน rate 1% มูลค่าขยับเป็นล้าน
  • 97% มาจาก reversion ปลายสัญญา → ถ้าสมมติฐาน land growth หรือการได้ทรัพย์คืนผิด มูลค่าผิดยกแผง

7. แบงค์ได้ประโยชน์อะไรจาก 2 กราฟนี้

📉 กราฟซ้าย — "หลักประกันเสื่อมค่าตามเวลายังไง"

ประโยชน์ต่อแบงค์อธิบาย
เห็นเพดานวงเงินกู้ ณ วันนี้เส้นประ ~6.95M = มูลค่าที่ปล่อยกู้ได้จริง ไม่ใช่ 15.5M ราคาตลาด → กัน over-lending
เห็นความเสี่ยงปลายสัญญาพื้นที่เขียวลาดลง → ยิ่งใกล้หมดสัญญา มูลค่ายิ่งหด ถ้า default ช่วงปลาย แบงค์ได้คืนน้อยลง
วางแผนระยะเวลากู้ควรให้สัญญากู้ ≤ อายุ leasehold ไม่ปล่อยกู้ยาวเกินอายุสิทธิการเช่า

📊 กราฟขวา — "มูลค่ามาจากไหน — ค่าเช่า vs การได้ทรัพย์คืน"

ประโยชน์ต่อแบงค์อธิบาย
รู้โครงสร้างความเสี่ยงRent ~3% / Reversion ~97% → มูลค่าเกือบทั้งหมดผูกกับการได้ทรัพย์คืนตอนจบ
ชี้จุดเปราะบางถ้าต่ออายุสัญญาไม่ได้/ไม่ได้ทรัพย์คืนตามคาด → 97% ของมูลค่าหายทันที ต้องตรวจเงื่อนไขสัญญาเช่าเข้ม
เส้น Cumulative PV%มูลค่าค่าเช่าสะสมเร็วช่วงต้น ช้าช่วงปลาย → ปลายสัญญาความคุ้มครองมาจาก reversion ล้วนๆ
🎯 ประโยชน์รวมที่แบงค์ได้:
  1. ตัดสินใจวงเงินกู้ — ใช้ ~6.95M (ไม่ใช่ 15.5M) เป็นฐาน LTV กัน over-lending
  2. ประเมินความเสี่ยง collateral — รู้ว่า 97% ผูกกับ reversion → ตรวจสัญญาเช่า/เงื่อนไขการคืนทรัพย์เข้มเป็นพิเศษ
  3. กำหนดเงื่อนไขสินเชื่อ — อายุกู้ ≤ อายุ leasehold, อาจขอหลักประกันเสริมถ้าปลายสัญญาเสี่ยง
  4. ทดสอบ sensitivity — แถบ Discount Rate (4%→8% ให้ 8.16M→4.89M) บอกว่าราคาไหวตัวแรงตาม rate → เลือก rate แบบ conservative ได้
  5. เอกสารประกอบการอนุมัติ — กราฟทำให้คณะกรรมการสินเชื่อเห็นภาพ "ทำไม leasehold ถึงมีค่าน้อยกว่าราคาตลาด" ในแผ่นเดียว

สรุปสั้น: กราฟซ้ายตอบ "ปล่อยกู้ได้เท่าไหร่และมูลค่าหดยังไง" ส่วนกราฟขวาตอบ "มูลค่านี้เชื่อถือได้แค่ไหน เสี่ยงตรงไหน" — รวมกันคือเครื่องมือตัดสินใจปล่อยสินเชื่อบนหลักประกัน leasehold อย่างมีข้อมูลครบ ไม่ประเมินสูงเกินจริง
